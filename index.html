How to Make a CSS Ribbon

Making ribbons in the browser is a play on the symmetrical and layered nature of CSS (e.g. flex, border, rgba), that can be manipulate to generate flexible stripes on a surface.

Create a ribbon structure


    <!-- a ribbon is a `figure` element, nesting 3 `span` elements -->

    <figure class="war some-war">
        <span class="outer"></span>
        <span class="inner"></span>
        <span class="outer"></span>
    </figure>

Add a common layout


    /*  common ribbon layout uses flex for symmetry purposes */

    .war {
        position: relative;
        overflow: hidden;
        display: flex;
        justify-content: center;
        width: 151px;
        height: 38px;
        box-shadow: 0 0 15px rgba(0, 0, 0, .2);
        border: 1px solid rgba(0, 0, 0, 0.5);
        border-radius: 3px;
        z-index: 1;
    }

Each ribbon is a overrides the ribbon layout and its inner parts


    /*  ribbon variant */

    .some-war {
        background: #c0a;
    }
    .some-war .outer {
        border-right: 12px double #fa9;
        border-left: 8px double rgba(100, 150, 200, 0.9);
        max-width: 8px;
    }
    .some-war .inner {
        background: #349;
        max-width: 44px;
    }
